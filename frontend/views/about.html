About EventSys

EventSys is your all-in-one platform for discovering, organizing, and joining events. Whether you're an organizer looking to publish and manage events, or a participant exploring new experiences, our platform offers a seamless and user-friendly solution.

This web application is built using modern technologies including HTML, CSS, JavaScript, Bootstrap, and PHP for the backend. It ensures responsiveness across all devices and supports both light and dark modes for better accessibility and user comfort.

Key features include user registration and login, event creation, browsing upcoming events, submitting reviews, and more. Administrators can manage users and events through a dedicated admin panel.
